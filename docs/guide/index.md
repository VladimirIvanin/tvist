# Введение

Tvist — слайдер для веб-приложений с широким API: опции, методы, события и подключаемые модули.

## Особенности

### 📖 Широкое API
Десятки опций настройки, методы управления, события жизненного цикла и навигации. Всё документировано — от базового слайдера до сложных сценариев с loop, thumbs и эффектами.

### 🎨 Гибкая стилизация
- BEM-нотация для классов
- SCSS переменные для кастомизации
- Готовые эффекты (fade, cube) и возможность создавать свои

### 🔧 Расширяемость
Подключаемые модули (навигация, пагинация, autoplay, thumbs и др.) и возможность добавлять свою функциональность.

## Доступные модули

- **Navigation** — стрелки навигации
- **Pagination** — точки-индикаторы
- **Autoplay** — автоматическое переключение
- **Drag** — свайп жестами
- **Loop** — бесконечная прокрутка
- **Thumbs** — миниатюры для навигации
- **Effects** — визуальные эффекты переходов (fade, cube)
